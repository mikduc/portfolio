# Portfolio

Personal portfolio website built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Overview

This project is a single-page, section-based portfolio with:
- Welcome, About, Projects, and Contact sections
- Smooth section scrolling and animated transitions
- Lazy-loaded sections for better initial load performance
- Light/dark theme support
- Supabase integration

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- Supabase (`@supabase/supabase-js`)
